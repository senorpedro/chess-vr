# VR Chessboard

Attempt to build a Chessboard in VR as web application using Aframe for WebXR, Svelte as framework and Typescript as language.

## Setup

`$ npm install`

## Run

`$ npm run dev`

Navigate WebXR able browser to **http://localhost:5000**
